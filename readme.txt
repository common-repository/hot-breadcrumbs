Hot Breadcrumbs widget shows a pathway (breadcrumbs) with link hierarchy of your site.

== Description ==

Hot Breadcrumbs widget shows a pathway (breadcrumbs) with link hierarchy of your site. This gives a better idea to the visitors how did they get to this page and what's the way back. Showing a link hierarchy also has positive SEO effects and allows better crawling of your site. This plugin uses the structured data to show breadcrumbs links.

== Installation ==

1. Install plugin as usually from WordPress.org. You can also manually download it, unpack and upload `plugin_hot_breadcrumbs` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Publish Hot Breadcrumbs widget into the available widget sidebars.

== Frequently Asked Questions ==

= Hot Breadcrumbs widget shows nothing. =

Check inner pages of your site. The widget shows empty output on home page. There's no need to use breadcrumbs links on home page. It's valuable on inner pages only.
